OpenCPN installieren

Für die Erstinstallation wird eine Internetverbindung benötigt. Navigieren Sie mit Ihrem Internetbrowser zu HIER und gehen Sie zum entsprechenden Abschnitt.

Wenn Sie während der Installation aufgefordert werden, ein Verzeichnis für Karten auszuwählen, überspringen Sie diesen Schritt. Wir behandeln Karten später HIER.

Windows

Laden Sie die Datei herunter. Führen Sie die .exe Datei aus und folgen Sie den Instruktionen.

Mac

Laden Sie die .pgk Datei herunter. Öffnen Sie die .pkg Datei und folgen Sie den Anweisungen im Installer.

Da das Programm nicht aus dem App-Store kommt, müssen Sie möglicherweise die Erlaubnis geben, von einem unbekannten Entwickler zu installieren.

System Einstellungen Security & Privacy

Raspberry Pi

Sie können OpenCPN installieren, indem Sie ein Ubuntu PPA (Personal Packages Archive) hinzufügen. Siehe Configuring Ubuntu PPA

Der einfachste Weg, OpenCPN auf dem Raspberry Pi zu installieren, ist OpenPlotter zu verwenden, welches OpenCPN und einige weitere Plugins installiert.

Ubuntu

Für Ubuntu Linux und seine Derivate (Linux Mint) und Debian Jessie basierte Distributionen wird OpenCPN aus einer PPA installiert. Configuring Ubuntu PPA erklärt, wie das geht.

Flatpak

OpenCPN Version 5.6.2 ist als Flatpak Paket verfügbar. Dieses Paketsystem wird von allen wichtigen Linux-Distributionen unterstützt. Vereinfachte Installationsanweisungen

--OR--

Manuelle Installation

  1. Installieren Sie flatpak auf Ihrem System wie in flatpak.org/setup beschrieben

  2. Installiere das OpenCPN Flatpak Image mit:

$ flatpak install --user  https://flathub.org/repo/appstream/org.opencpn.OpenCPN.flatpakref

OpenCPN starten:

$ flatpak run org.opencpn.OpenCPN

Fedora

OpenCPN wird auf Fedora nicht nativ unterstützt. Verwenden Sie stattdessen die Flatpak Release.

Debian

Für Debian-Derivate können Sie die gleichen Pakete wie für die gleichwertige Ubuntu-Veröffentlichung verwenden. Für Debian Stable (Buster) können Sie die Pakete von Ubuntu PPA verwenden. Das kompatible Ubuntu Release ist Bionic (Ubuntu 18.04).
