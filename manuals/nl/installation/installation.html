OpenCPN installeren

Voor de eerste installatie is een internetverbinding nodig. Met je internetbrowser ga je naar HIER en ga naar de relevante sectie.

Als tijdens de installatie gevraagd wordt om een map te selecteren, die de kaarten bevat, deze stap van het proces overslaan. Kaarten worden gedekt door HIER.

Windows

Download het .exe bestand. Open het .exe bestand en volg de instructies.

Mac

Download het .pkg bestand. Open .pkg en volg de instructies in het installatieprogramma.

Omdat het programma niet uit de app store komt, moet je mogelijk toestemming geven om te installeren van een onbekende ontwikkelaar.

Systeemvoorkeuren Veiligheid & Privacy

Raspberry Pi

Je kunt OpenCPN installeren door een PPA (Persoonlijk Pakketten Archief) van Ubuntu toe te voegen. Ubuntu PPA legt uit hoe dit wordt gebruikt.

De eenvoudigste manier om OpenCPN op de Raspberry Pi te installeren is door OpenPlotter te gebruiken die OpenCPN en ook enkele plugins installeert.

Ubuntu

Voor Ubuntu Linux en zijn afgeleiden (Linux Mint) en op Debian Jessie gebaseerde distributies, wordt OpenCPN verspreid vanuit een PPA. Ubuntu PPA legt uit hoe dit wordt gebruikt.

Flatpak

OpenCPN Versie 5.6.2 is beschikbaar als Flatpak pakket. Dit verpakkingssysteem wordt ondersteund bij alle grote Linux-distributies. Vereenvoudigde installatie-instructies

--Of--

Handmatige installatie

  1. Installeer flatpak op uw systeem zoals beschreven in flatpak.org/setup

  2. Installeer OpenCPN Flatpak afbeelding met gebruik:

$ flatpak installatie --user https://flathub.org/repo/appstream/org.opencpn.OpenCPN.flatpakref

Start OpenCPN:

$ flatpak run org.opencpn.OpenCPN

Fedora

OpenCPN wordt niet ondersteund op Fedora. Gebruik in plaats daarvan de Flatpak-versie.

Debian

Voor Debian-derivaten kun je dezelfde pakketten gebruiken als voor de gelijkwaardige Ubuntu-release. Voor Debian Stable (Buster) kan je de pakketten gebruiken via Ubuntu PPA. De compatibele Ubuntu Release is Bionic (Ubuntu 18.04).
